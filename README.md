# Console Input Handling in C++ (Windows Compatible)

This project demonstrates **secure character input handling** in the console using the `<conio.h>` library in C++.  
It allows taking user input **without displaying characters on the screen**, which is especially useful for **password input** or **interactive command-line applications**.

---

## Features

- Hides input characters while typing (secure password entry).
- Reads each keystroke instantly using `_getch()` (no need to press Enter).
- Supports backspace functionality for correction.
- Lightweight, clean, and fully **Windows-compatible**.
- Simple integration into any C++ console project.

---

## Requirements

- **Compiler:** MinGW / MSVC  
- **IDE:** Visual Studio Code or any Windows-based C++ IDE  
- **Operating System:** Windows 10 / 11
